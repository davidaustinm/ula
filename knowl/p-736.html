We have the augmented matrix
\begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 2 \\ 1 \amp 1 \amp 1 \amp 3 \\ 1 \amp 2 \amp 4 \amp 0 \\ 1 \amp 3 \amp 9 \amp 1 \\ \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array}\right]\text{.} \end{equation*}
This system is therefore inconsistent, which means there is no degree 2 polynomial passing through the four points.
