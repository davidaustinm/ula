  2. We have \(\det(A^3) = (\det(A))^3 = -8\text{.}\)
  3. We know that \(\det(AB) = \det(A) \det(B) = -10\text{.}\)
  4. Multiplying the matrix by \(-1\) scales each row by \(-1\) so we have \(\det(-A) = (-1)^5\det(A) = 2\text{.}\)
  5. \(\det(AB^{-1}) = \det(A) \det(B^{-1}) = \det(A)/\det(B) = -\frac 25\text{.}\)
