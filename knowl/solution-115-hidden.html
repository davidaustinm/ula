1. The matrix \(\left[\begin{array}{rr} r \amp 0 \\ 0 \amp r \\ \end{array}\right]\) stretches vectors by \(r\text{.}\) It has the same effect as scalar multiplication by \(r\text{.}\)
  2. This matrix rotates vectors by an angle \(\theta\text{.}\)
  3. No matter which order we multiply the matrices, we find that their product is
    \begin{equation*} \left[\begin{array}{rr} r \amp 0 \\ 0 \amp r \\ \end{array}\right] \left[\begin{array}{rr} \cos\theta \amp -\sin\theta \\ \sin\theta \amp \sin\theta \\ \end{array}\right] = \left[\begin{array}{rr} r\cos\theta \amp -r\sin\theta \\ r\sin\theta \amp r\sin\theta \\ \end{array}\right]\text{.} \end{equation*}
  4. This follows from the previous part.
  5. If we use polar coordinates to write \(\twovec{a}{b} = \twovec{r\cos\theta}{r\sin\theta}\text{,}\) multiplying a vector by \(A\) will stretch the vector by \(r\) and rotate it by \(\theta\text{.}\)
  6. If
    \begin{equation*} \twovec{a}{b} = \twovec{r\cos\theta}{r\sin\theta}, \twovec{c}{d} = \twovec{s\cos\phi}{s\sin\phi}\text{,} \end{equation*}
    then \(S\circ T\) has the effect of stretching a vector by \(rs\) and rotating it by \(\theta + \phi\text{.}\)
