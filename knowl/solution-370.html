Solution 7.1.4.4.1.

  1. Because \(B^T = (A^TA)^T = A^TA = B\text{,}\) the matrix \(B\) is symmetric and hence orthogonally diagonalizable.
  2. \(\vvec\cdot(B\vvec) = \vvec\cdot(A^TA\vvec) = (A\vvec)\cdot(A\vvec) = |A\vvec|^2\text{.}\)
  3. \(\displaystyle |A\uvec|^2 = \uvec\cdot(B\uvec) = \lambda\uvec\cdot\uvec = \lambda\)
  4. Because \(\lambda=|A\uvec|^2 \geq 0\text{.}\)
  5. In the same way, \(C=\frac1N AA^T\) so that \(\uvec\cdot(C\uvec) = \frac1N(A^T\uvec)\cdot(A^T\uvec) = \frac1N |A^T\uvec|^2\text{.}\) Therefore, \(\lambda = \frac1N|A^T\uvec|^2\text{.}\)
