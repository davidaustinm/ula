1. The pivot positions are indicated below
    \begin{equation*} \left[ \begin{array}{rrrr} {\mathbf 2} \amp 4 \amp 6 \amp -1 \\ -3 \amp {\mathbf 1} \amp 5 \amp 0 \\ 1 \amp 3 \amp 5 \amp {\mathbf 1} \\ \end{array} \right] \sim \left[ \begin{array}{rrrr} {\mathbf 1} \amp 0 \amp -1 \amp 0 \\ 0 \amp {\mathbf 1} \amp 2 \amp 0 \\ 0 \amp 0 \amp 0 \amp {\mathbf 1} \\ \end{array} \right]\text{.} \end{equation*}
  2. A row contains at most one pivot position. Therefore, a \(3\times 5\) matrix, which has three rows, contains at most three pivot positions. Here is an example:
    \begin{equation*} \left[\begin{array}{rrrrr} 1 \amp 0 \amp 0 \amp -1 \amp 2 \\ 0 \amp 1 \amp 0 \amp 1 \amp 3 \\ 0 \amp 0 \amp 1 \amp 2 \amp 3 \\ \end{array}\right]\text{.} \end{equation*}
  3. A column contains at most one pivot position. Therefore, a \(5\times 3\) matrix, which has three columns, contains at most three pivot positions. Here is an example
    \begin{equation*} \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \\ \end{array}\right]\text{.} \end{equation*}
  4. A matrix with a pivot position in every row and every column would have the following reduced row echelon form:
    \begin{equation*} \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right]\text{.} \end{equation*}
    Such a matrix must necessarily have the same number of rows and columns, which means it is what we call a square matrix.
