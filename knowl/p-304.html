Give an example of a reduced row echelon matrix that describes a linear system having the stated properties. If it is not possible to find such an example, explain why not.
  1. Write a reduced row echelon matrix for a linear system having five equations and three variables and having exactly one solution.
  2. Write a reduced row echelon matrix for a linear system having three equations and three variables and having no solution.
  3. Write a reduced row echelon matrix for a linear system having three equations and five variables and having infinitely many solutions.
  4. Write a reduced row echelon matrix for a linear system having three equations and four variables and having exactly one solution.
  5. Write a reduced row echelon matrix for a linear system having four equations and four variables and having exactly one solution.
