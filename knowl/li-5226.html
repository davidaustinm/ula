Item 7.2.4.5.c.
The minimum value of \(0\) says there is a vector such that \(q(\uvec) = |B\uvec|^2 = 0\text{,}\) which means that \(B\uvec=0\text{.}\) So there is a nonzero vector that is a solution to the homogeneous equations, which implies that the columns of \(B\) are linearly dependent.
