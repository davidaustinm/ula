  1. The maximum appears to occur in the direction of \(\twovec10\) and the minimum appears to occur in the direction of \(\twovec01\text{.}\)
  2. \(q_D(\xvec) = 3x_1^2 - x_2^2\) so that \(q_D\left(\twovec10\right) = 3\) and \(q_D\left(\twovec01\right) = -1\text{.}\)
  3. We have \(q_D(\uvec) = 3u_1^2 - u_2^2 = 3 - 4u_2^2\) so that the quadratic form only depends on \(u_2\text{.}\) The graph of this function of \(u_2\) is a parabola that has a maximum of \(3\) when \(u_2=0\text{.}\) Since \(u_1^2 = 1-u_2^2\text{,}\) this means that the maximum occurs when \(u_1=\pm 1\text{.}\) We therefore see that the maximum value of \(q_D(\uvec)\) is \(3\) in the direction \(\twovec10\) as we saw from the graph.
  4. Now \(q_D(\uvec) = -1 + 4u_1^2\text{,}\) which has a minimum value of \(-1\) when \(u_1=0\text{.}\) Therefore, the minimum value of \(q_D(\uvec)\) is \(-1\) in the direction \(\twovec01\text{.}\)
  5. The graph of \(q_A\) appears to be similar to the graph of \(q_D\) only rotated by \(45^\circ\text{.}\) This means the maximum appears to occur in the direction \(\twovec11\) and the minimum in the direction \(\twovec{-1}1\text{.}\)
  6. Since \(Q\) is orthogonal, we have \(QQ^T=I\) so that
    \begin{equation*} |\vvec|^2 = (Q^T\uvec)\cdot(Q^T\uvec) = \uvec\cdot(QQ^T\uvec) = \uvec\cdot\uvec = |\uvec|^2 = 1\text{.} \end{equation*}
  7. Since \(q_A(\uvec) = q_D(\vvec)\text{,}\) the maximum of \(q_A(\uvec)\) is \(3\text{,}\) which occurs when \(\vvec = Q^T\uvec = \twovec10\text{.}\) This means that \(\uvec=Q\twovec10 = \twovec{1/\sqrt{2}}{1/\sqrt{2}}\text{,}\) the eigenvector of \(A\) associated to \(\lambda=3\text{.}\)
    In the same way, the minimum value of \(q_A(\uvec)\) is \(-1\text{,}\) which occurs when \(\uvec=\twovec{1/\sqrt{2}}{1/\sqrt{2}}\text{,}\) the eigenvector of \(A\) associated to \(\lambda=-1\text{.}\)
