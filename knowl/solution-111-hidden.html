We construct the matrices as \(\left[\begin{array}{rrr} T(\evec_1) \amp T(\evec_2) \amp T(\evec_3) \\ \end{array}\right]\text{.}\)
  1. \(\left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp -1 \\ 0 \amp 1 \amp 0 \\ \end{array}\right] \text{.}\)
  2. \(\left[\begin{array}{rrr} 0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \\ -1 \amp 0 \amp 0 \\ \end{array}\right] \text{.}\)
  3. \(\left[\begin{array}{rrr} 0 \amp -1 \amp 0 \\ 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\)
  4. \(\left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \\ 0 \amp -1 \amp 0 \\ \end{array}\right] \left[\begin{array}{rrr} 0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \\ -1 \amp 0 \amp 0 \\ \end{array}\right] \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp -1 \\ 0 \amp 1 \amp 0 \\ \end{array}\right]= \left[\begin{array}{rrr} 0 \amp 1 \amp 0 \\ -1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{,}\) which is a \(-90^\circ\) rotation about the \(z\)-axis.
