Solution 7.1.4.9.1.

Since \(A\) is not invertible, the third eigenvalue must be zero: \(\lambda_3=0\text{.}\) Also, an eigenvector associated to \(\lambda_3\) must be orthogonal to both \(\vvec_1\) and \(\vvec_2\text{.}\) We can find such a vector by finding \(\nul(B^T)\) where \(B=\begin{bmatrix}\vvec_1\amp\vvec_2\end{bmatrix}\text{.}\) This leads to
\begin{equation*} D = \begin{bmatrix} 20 \amp 0 \amp 0 \\ 0 \amp -4 \amp 0 \\ 0 \amp 0 \amp 0 \\ \end{bmatrix},\hspace{24pt} Q = \begin{bmatrix} 2/3 \amp 1/\sqrt{18} \amp -1/\sqrt{2} \\ -1/3 \amp 4/\sqrt{18} \amp 0 \\ 2/3 \amp 1/\sqrt{18} \amp 1/\sqrt{2} \\ \end{bmatrix} \end{equation*}
