1. The linear combination
    \begin{equation*} 2\vvec_1-3\vvec_2+\vvec_3 = \twovec{5}{-1}\text{.} \end{equation*}
  2. The appropriate linear system corresponds to the augmented matrix
    \begin{equation*} \left[\begin{array}{rrr|r} 2 \amp -1 \amp -2 \amp 0 \\ 1 \amp 1 \amp 0 \amp 0 \\ \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp -\frac{2}{3} \amp 0 \\ 0 \amp 1 \amp \frac{2}{3} \amp 0 \\ \end{array}\right]\text{.} \end{equation*}
    This shows we obtain \({\mathbf 0}\) when the weights are chosen so that
    \begin{equation*} \begin{alignedat}{2} c_1 \amp {}={} \frac23 c_3 \\ c_2 \amp {}={} -\frac23 c_3\text{.} \\ \end{alignedat} \end{equation*}
  3. In this case, we want \(c_3=0\text{,}\) which means that \(c_1=c_2=0\text{.}\)
  4. We see that \(\vvec_3=-\frac23\vvec_1 + \frac23\vvec_2\text{.}\)
