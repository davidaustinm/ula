Item.

A solution to the linear system whose augmented matrix is
\begin{equation*} \left[\begin{array}{rrrr|r} \vvec_1 \amp \vvec_2 \amp \ldots \amp \vvec_n \amp \bvec \end{array}\right] \end{equation*}
is a set of weights that expresses \(\bvec\) as a linear combination of \(\vvec_1,\vvec_2,\ldots,\vvec_n\text{.}\)
