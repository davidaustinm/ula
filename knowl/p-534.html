A column contains at most one pivot position. Therefore, a \(5\times 3\) matrix, which has three columns, contains at most three pivot positions. Here is an example
\begin{equation*} \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \\ \end{array}\right]\text{.} \end{equation*}
