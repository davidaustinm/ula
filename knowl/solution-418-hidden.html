1. If the columns are linearly independent, then \(\nul(A)=\{\zerovec\}\text{,}\) which means that the rank equals the number of columns. Therefore, every column of \(\Sigma\) has a nonzero singular value.
  2. In the columns span \(\real^m\text{,}\) then \(\col(A)=\real^m\text{,}\) which says that the rank equals the number of rows. Therefore, every row of \(\Sigma\) has a nonzero singular value.
  3. Then \(\Sigma\) is a square matrix and every singular value is nonzero.
