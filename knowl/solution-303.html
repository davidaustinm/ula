Solution 6.2.3.1.

  1. \(\displaystyle A^T = \begin{bmatrix} 3 \amp -1 \amp 0 \\ 4 \amp 2 \amp -2 \end{bmatrix}\)
  2. \begin{align*} \vvec_1\cdot\wvec \amp{}={} 2(-2) + 0(2) + (-2)3 \amp{}={} -10\\ \vvec_2\cdot\wvec \amp{}={} 1(-2) + 1(2) + 2(3) \amp{}={} 6 \end{align*}
  3. \begin{equation*} A^T\wvec = \twovec{2(-2)+0(2)+(-2)3}{1(-2)+1(2)+2(3)} = \twovec{-10}{6} =\twovec{\vvec_1\cdot\wvec}{\vvec_2\cdot\wvec} \end{equation*}
  4. Both dot products are 0 so we have \(A^T\xvec = \zerovec\text{.}\)
  5. We need to solve the equation \(A^T\xvec = \zerovec\) so we find the reduced row echelon form
    \begin{equation*} A^T \sim \begin{bmatrix} 1 \amp 0 \amp -1 \\ 0 \amp 1 \amp 3\\ \end{bmatrix}\text{.} \end{equation*}
    The vectors orthogonal to both \(\vvec_1\) and \(\vvec_2\) have the form \(\xvec=x_3\threevec1{-3}1\text{.}\)
  6. \(A^T\xvec = \twovec{\vvec_1\cdot\xvec}{\vvec_2\cdot\xvec} = \twovec00\) tells us that \(\vvec_1\cdot\xvec=0\) and \(\vvec_2\cdot\xvec=0\text{.}\)
  7. Since \(\xvec\) is orthogonal to both \(\vvec_1\) and \(\vvec_2\text{,}\) we have
    \begin{equation*} (c_1\vvec_1+c_2\vvec_2)\cdot\xvec = c_1\vvec_1\cdot\xvec + c_2\vvec_2\cdot\xvec = 0 \end{equation*}
