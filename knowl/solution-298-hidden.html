1. If \(\vvec_3 = c_1\vvec_1 + c_2\vvec_2\text{,}\) then \(c_1 = \frac{\vvec_3\cdot\vvec_1}{\vvec_1\cdot\vvec_2} = 0\text{.}\) In the same way, \(c_2=0\) so \(\vvec_3\) must be the zero vector. We’re told, however, that \(\vvec_3\) is nonzero.
  2. We’ve seen that \(\vvec_3\) is not a linear combination of \(\vvec_1\) and \(\vvec_2\text{.}\) The same thinking shows that none of the vectors is a linear combination of the others so they form a linearly independent set.
