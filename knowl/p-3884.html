A vector lying along the line of reflection is unchanged so \(A\twovec{1}{1} = \twovec{1}{1}\text{,}\) which shows that \(\twovec{1}{1}\) is an eigenvector with associated eigenvalue \(\lambda = 1\text{.}\) At the same time, \(A\twovec{1}{-1} = -\twovec{1}{-1}\) so \(\twovec{1}{-1}\) is an eigenvector with associated eigenvalue \(\lambda=-1\text{.}\)
