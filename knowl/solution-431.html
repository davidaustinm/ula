Solution 7.5.7.7.1.

  1. The number of columns of \(A^+\) is the number of columns of \(U_r^T\text{,}\) which is \(m\text{,}\) and the number of rows is the number of rows of \(V_r\text{,}\) which is \(n\text{.}\)
  2. If \(A\) is invertible, then \(A\) is an \(m\times m\) matrix whose rank is \(m\text{.}\) The reduced singular value decomposition is therefore \(A=U\Sigma V^T\) so that \(A^+=V\Sigma^{-1}U^T = (U\Sigma V^T)^{-1}=A^{-1}\text{.}\)
  3. \(AA^{+} = (U_r\Sigma_rV_r^T)(V_r\Sigma^{-1}U_r^T) = U_rU_r^T\text{,}\) which projects vectors orthogonal onto \(\col(A)\) since the columns of \(U_r\) are an orthonormal basis for \(\col(A)\text{.}\)
  4. \(AA^{+} = (V_r\Sigma^{-1}U_r^T)(U_r\Sigma_rV_r^T) V_rV_r^T\text{,}\) which projects vectors orthogonal onto \(\col(A^T)\) since the columns of \(V_r\) are an orthonormal basis for \(\col(A^T)\text{.}\)
