A matrix with a pivot position in every row and every column would have the following reduced row echelon form:
\begin{equation*} \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right]\text{.} \end{equation*}
Such a matrix must necessarily have the same number of rows and columns, which means it is what we call a square matrix.
