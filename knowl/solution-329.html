Solution 6.3.4.10.1.

  1. If this matrix is \(Q\text{,}\) we have \(Q^TQ=I\text{.}\)
  2. If this matrix is \(Q\text{,}\) we have \(Q^TQ=I\text{.}\)
  3. \(\uvec_2=\twovec{-\sin\theta}{\cos\theta}\) or \(\twovec{\sin\theta}{-\cos\theta}\)
  4. If \(Q=\begin{bmatrix}\uvec_1\amp\uvec_2\end{bmatrix}\) is an orthogonal \(2\times2\) matrix, then \(\uvec_1\) is a unit vector and has the form \(\uvec_1=\twovec{\cos\theta}{\sin\theta}\) for some angle \(\theta\text{.}\) By the last part of this problem, there are only two choices for \(\uvec_2\text{,}\) one of which gives a rotation and one of which gives a reflection.
