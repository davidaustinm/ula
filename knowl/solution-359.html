Solution 6.5.6.11.1.

  1. Starting with the assumption that \(A^TA\xvec = \zerovec\text{,}\) we dot both sides of the equation with \(\xvec\) to obtain \(\xvec \cdot A^TA\xvec = 0\text{.}\) From here, we write
    \begin{equation*} \begin{aligned} \xvec\cdot A^TA\xvec \amp = 0 \\ \xvec^TA^TA\xvec \amp = 0 \\ (A\xvec)^T(A\xvec) \amp = 0 \\ (A\xvec)\cdot (A\xvec) \amp = 0 \\ \len{A\xvec}^2 \amp = 0. \\ \end{aligned} \end{equation*}
    This says that the length of \(A\xvec\) must be zero, which tells us that \(A\xvec=\zerovec\text{.}\)
  2. If the columns of \(A\) are linearly independent, we know that the only solution to the homogeneous equation \(A\xvec = \zerovec\) is \(\xvec = \zerovec\text{.}\)
  3. The previous two parts of this exercise tell us that \(A^TA\xvec = \zerovec\) implies that \(A\xvec = \zerovec\text{,}\) which implies that \(\xvec = \zerovec\text{.}\)
  4. Since the only solution to the homogeneous equation \(A^TA\xvec = \zerovec\) is \(\xvec=\zerovec\text{,}\) the columns of \(A^TA\) are linearly independent, which means that \(A^TA\xhat=A^T\bvec\) has only one solution.
