Item i.
If we apply a sequence of row operations to \(A\) to find a row equivalent matrix \(U\) that is upper triangular, we know that \(\det(A)\neq0\) if and only if \(\det(U) \neq 0\text{.}\) We also know that \(A\) is invertible if and only if \(U\) is invertible. Putting these facts together, we conclude that \(\det(A)\neq 0\) if and only if \(A\) is invertible.
