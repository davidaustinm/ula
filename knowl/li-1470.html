Item d.
The character is successively rotated by \(45^\circ\) using the matrix \(\left[\begin{array}{rrr} \cos(45^\circ) \amp -\sin(45^\circ) \amp 0 \\ \sin(45^\circ) \amp \sin(45^\circ) \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] = \left[\begin{array}{rrr} \frac{1}{\sqrt{2}} \amp -\frac{1}{\sqrt{2}} \amp 0 \\ \frac{1}{\sqrt{2}} \amp \frac{1}{\sqrt{2}} \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\)
