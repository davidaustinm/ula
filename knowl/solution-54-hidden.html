1. We construct the augmented matrix
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 0 \amp 2 \amp 0 \\ 2 \amp 2 \amp 2 \amp 0 \\ -1 \amp -3 \amp 1 \amp 0 \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 2 \amp 0 \\ 0 \amp 1 \amp -1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \end{array}\right]\text{,} \end{equation*}
    which shows that
    \begin{equation*} \xvec=\threevec{-2x_3}{x_3}{x_3} = x_3\threevec{-2}{1}{1}\text{.} \end{equation*}
  2. We form \(B\) by choosing two vectors in the solution space to \(A\xvec=\zerovec\text{.}\) For instance, \(B = \left[\begin{array}{rr} -2 \amp 2 \\ 1 \amp -1 \\ 1 \amp -1 \\ \end{array}\right]\text{.}\)
