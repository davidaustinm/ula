Item 2.1.4.3.b.
The appropriate linear system corresponds to the augmented matrix
\begin{equation*} \left[\begin{array}{rrr|r} 2 \amp -1 \amp -2 \amp 0 \\ 1 \amp 1 \amp 0 \amp 0 \\ \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp -\frac{2}{3} \amp 0 \\ 0 \amp 1 \amp \frac{2}{3} \amp 0 \\ \end{array}\right]\text{.} \end{equation*}
This shows we obtain \({\mathbf 0}\) when the weights are chosen so that
