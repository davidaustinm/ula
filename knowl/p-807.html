This is the same as asking if the linear system corresponding to the following augmented matrix is consistent:
\begin{equation*} \left[\begin{array}{rrr|r} 0 \amp 1 \amp 2 \amp -1 \\ -2 \amp 1 \amp 0 \amp 3 \\ 1 \amp -1 \amp -1 \amp -1 \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 1 \amp -2 \\ 0 \amp 1 \amp 2 \amp -1 \\ 0 \amp 0 \amp 0 \amp 0 \end{array}\right]\text{.} \end{equation*}
From the reduced row echelon form, we see that the system is consistent, which means that \(\bvec\) can be expressed as a linear combination of \(\vvec_1\text{,}\) \(\vvec_2\text{,}\) and \(\vvec_3\text{.}\) Moreover, there are infinitely ways in which we can do so.
