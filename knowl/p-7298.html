  1. If \(Q\) is a \(3\times2\) matrix whose columns are an orthonormal basis for \(W\text{,}\) then \(P=QQ^T\text{.}\) This means that \(P^T=(Q^T)^TQ^T = QQ^T = P\) so \(P\) is symmetric and hence orthogonally diagonalizable.
  2. If \(\vvec\) is in \(W\text{,}\) then \(P\vvec=\vvec\) and if \(\vvec\) is in \(W^\perp\text{,}\) then \(P\vvec=0\text{.}\) This means that the eigenvalues of \(P\) are either 0 or 1.
  3. The eigenspaces \(E_1=W\) and \(E_0=W^{\perp}\)
