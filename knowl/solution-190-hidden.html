1. If \(\lambda=0\) is an eigenvalue, then there is a nonzero vector \(\vvec\) such that \(A\vvec=\lambda\vvec = \zerovec\text{.}\) This means that an associated eigenvector is a nonzero solution to the homogeneous equation \(A\xvec=\zerovec\text{.}\)
  2. If \(\lambda =0\) is an eigenvalue of \(A\text{,}\) then there is a nonzero solution to the homogeneous equation \(A\xvec=\zerovec\text{,}\) which happens exactly when \(A\) is not invertible.
  3. If \(\vvec\) is an eigenvector of \(A\) with associated eigenvalue \(\lambda\text{,}\) then \(A\vvec=\lambda\vvec\text{.}\) Therefore, \(A^2\vvec = A(A\vvec) = \lambda A\vvec = \lambda^2\vvec\text{,}\) which means that \(\vvec\) is an eigenvector with associated eigenvalue \(\lambda^2\text{.}\)
  4. If \(A\vvec=\lambda\vvec\text{,}\) we can multiply both sides by \(A^{-1}\) and \(\frac1\lambda\) to obtain \(\frac1\lambda\vvec=A^{-1}\vvec\text{.}\) This shows that \(\vvec\) is an eigenvector of \(A^{-1}\) with associated eigenvalue \(\frac1\lambda\text{.}\)
  5. The vectors \(\vvec_1\) and \(\vvec_2\) are eigenvectors of \(A^5\) with associated eigenvalues \(\lambda_1^5 = 3^5=243\) and \(\lambda_2^5 = (-1)^5 = -1\text{.}\)
