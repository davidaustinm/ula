Item 7.5.7.4.e.
The first principal component is \(\uvec_1 = \fourvec{-0.64}{-0.20}{0.36}{-0.65}\text{.}\) If we move to the left, this vector is multiplied by a negative number so the living area, lot size, and price go up while the age of the house decreases.
The second principal component is \(\uvec_2=\fourvec{0.05}{0.86}{0.51}{-0.03}\text{.}\) If we move up, this is multiplied by a positive number so the lot size and the age of the house increase.
