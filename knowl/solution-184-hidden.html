1. We have
    \begin{equation*} A = \left[\begin{array}{rr} 2 \amp -4 \\ -1 \amp 2 \\ \end{array}\right] \sim \left[\begin{array}{rr} 1 \amp -2 \\ 0 \amp 0 \\ \end{array}\right]\text{,} \end{equation*}
    which shows that \(\xvec=x_2\twovec{2}{1}\) describes the solution space to the homogeneous equation.
  2. The solution space forms a line through the origin.
  3. The solution space has the parametric description \(\xvec=\twovec{2}{0}+x_2\twovec{2}{1}\text{,}\) which is a line parallel to the solution space to the homogeneous equation.
  4. This cannot form a subspace since it does not contain the vector \(\zerovec\text{.}\)
  5. The solution space has the parametric description \(\xvec=\twovec{-4}{0}+x_2\twovec{2}{1}\text{,}\) which is a line parallel to the solution space to the homogeneous equation. The graph of the solution spaces is shown below.
  6. They are all parallel to the solution space to the homogeneous equation.
  7. If the solution space is a subspace, then \(\xvec=\zerovec\) must be in the solution space. This means that \(A\xvec = A\zerovec = \zerovec = \bvec\text{.}\)
