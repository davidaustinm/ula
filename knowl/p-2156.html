The composition of this reflection with itself is described by multiplying the matrix by itself. This produces the matrix \(\left[\begin{array}{rr} 1 \amp 0 \\ 0 \amp 1 \\ \end{array}\right] \text{,}\) which we just saw is the matrix for the identity transformation. This means that reflecting a vector in the line \(y=x\) twice produces the original vector.
