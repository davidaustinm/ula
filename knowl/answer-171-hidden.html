1. If \(A=PBP^{-1}\text{,}\) then \(B=P^{-1}AP\text{.}\) If we call \(Q=P^{-1}\text{,}\) then \(B=QAQ^{-1}\text{,}\) which shows that \(B\) is similar to \(A\text{.}\)
  2. If \(A=PBP^{-1}\) and \(B=QCQ^{-1}\text{,}\) then \(A=PQCP^{-1}Q^{-1} = (PQ)C(PQ)^{-1}\text{,}\) which shows that \(A\) is similar to \(C\text{.}\)
  3. If \(B\) is diagonalizable, then it is similar to a diagonal matrix \(D\text{.}\) Since \(A\) is similar to \(B\) and \(B\) is similar to \(D\text{,}\) we know that \(A\) is also similar to \(D\) so that \(A\) is diagonalizable.
  4. If \(A=PBP^{-1}\text{,}\) then
    \begin{equation*} \begin{aligned} \det(A-\lambda I) \amp = \det(PBP^{-1}-\lambda PP^{-1}) \\ \amp = \det[P(B-\lambda I)P^{-1}] \\ \amp = \det(B-\lambda I)\text{.} \\ \end{aligned} \end{equation*}
  5. The eigenvalues of a matrix are the roots of its characteristic polynomial. If \(A\) and \(B\) are similar, then they have the same characteristic polynomial and hence the same eigenvalues.
