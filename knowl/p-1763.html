If \(\wvec\) is a linear combination of \(\vvec_1\) and \(\vvec_2\text{,}\) then we can write \(\wvec=a\vvec_1+b\vvec_2\text{.}\) Then any linear combination of \(\vvec_1,\vvec_2,\wvec\) can be rewritten as
\begin{equation*} c_1\vvec_1+c_2\vvec_2+c_3\wvec=(c_1+c_3a)\vvec_1 + (c_2+c_3b)\vvec_2\text{.} \end{equation*}
