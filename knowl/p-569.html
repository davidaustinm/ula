The third equation, however, says that
\begin{equation*} 0x + 0y + 0z = 1\text{.} \end{equation*}
Again, if we evaluate the left-hand side with any values of \(x\text{,}\) \(y\text{,}\) and \(z\text{,}\) we get 0 so this equation cannot be satisfied for any \((x,y,z)\text{.}\) This means that the entire linear system has no solution and is therefore inconsistent.
