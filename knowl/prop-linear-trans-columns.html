Proposition 2.5.6.

If \(T:\real^n\to\real^m\) is a matrix transformation given by \(T(\xvec) = A\xvec\text{,}\) then the matrix \(A\) has columns \(T(\evec_j)\text{;}\) that is,
\begin{equation*} A = \left[\begin{array}{rrrr} T(\evec_1) \amp T(\evec_2) \amp \cdots \amp T(\evec_n) \end{array}\right]\text{.} \end{equation*}
