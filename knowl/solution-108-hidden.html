1. \begin{equation*} \left[\begin{array}{rrr} a \amp b \amp c \\ d \amp e \amp f \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \threevec{x}{y}{1} = \cthreevec{ax+by+c}{dx+ey+f}{1}\text{,} \end{equation*}
    which shows that
    \begin{equation*} \begin{alignedat}{2} x' \amp {}={} ax+by+c \\ y' \amp {}={} dx+ey+f\text{.} \\ \end{alignedat} \end{equation*}
  2. Notice that the entries \(c\) and \(f\) are responsible for the translation. Therefore, we need the matrix transformation defined by \(\left[\begin{array}{rrr} 1 \amp 0 \amp 2 \\ 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\)
  3. We would like to reflect in the vertical axis so we use the matrix \(\left[\begin{array}{rrr} -1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\)
  4. The character is successively rotated by \(45^\circ\) using the matrix \(\left[\begin{array}{rrr} \cos(45^\circ) \amp -\sin(45^\circ) \amp 0 \\ \sin(45^\circ) \amp \sin(45^\circ) \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] = \left[\begin{array}{rrr} \frac{1}{\sqrt{2}} \amp -\frac{1}{\sqrt{2}} \amp 0 \\ \frac{1}{\sqrt{2}} \amp \frac{1}{\sqrt{2}} \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\)
  5. We first translate the character down two units using the matrix \(\left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp -2 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\) Then we zoom in by uniformly stretching by a factor of \(2\) using the matrix \(\left[\begin{array}{rrr} 2 \amp 0 \amp 0 \\ 0 \amp 2 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\) The net effect is the transformation described by the matrix
    \begin{equation*} \left[\begin{array}{rrr} 2 \amp 0 \amp 0 \\ 0 \amp 2 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \left[\begin{array}{rrr} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp -2 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] = \left[\begin{array}{rrr} 2 \amp 0 \amp 0 \\ 0 \amp 2 \amp -4 \\ 0 \amp 0 \amp 1 \\ \end{array}\right]\text{.} \end{equation*}
  6. The shadow is first created using the shear \(\left[\begin{array}{rrr} 1 \amp 1 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\) Then the vertical scale is compressed using the matrix \(\left[\begin{array}{rrr} \frac12 \amp 0 \amp 0 \\ 0 \amp \frac12 \amp 0 \\ 0 \amp 0 \amp 1 \\ \end{array}\right] \text{.}\)
