1. The components of the vector \(\coords{\xvec}{\bcal}\) are the weights that express \(\xvec\) as a linear combination of the basis vectors; that is, \(\coords{\xvec}{\bcal} = \cfourvec{c_1}{c_2}{\vdots}{c_m}\) if \(\xvec=c_1\vvec_1+c_2\vvec_2+\ldots+c_m\vvec_m\text{.}\)
  2. If we form the matrix \(P_{\bcal} = \left[\begin{array}{rrrr} \vvec_1 \amp \vvec_2 \amp \ldots \amp \vvec_m \end{array}\right]\text{,}\) then \(\xvec=P_{\bcal}\coords{\xvec}{\bcal}\text{.}\)
  3. As before, \(\coords{\xvec}{\bcal}=P_{\bcal}^{-1}\xvec\text{.}\)
  4. We find \(\xvec=P_{\bcal}\coords{\xvec}{\bcal} = \twovec{-1}{1}\text{.}\)
  5. We find \(\coords{\xvec}{\bcal}=P_{\bcal}^{-1}\xvec = \twovec{-3}{5}\text{.}\)
