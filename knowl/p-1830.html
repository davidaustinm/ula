More generally, we will write the columns of the \(n\times n\) identity matrix as
\begin{equation*} \evec_1 = \left[\begin{array}{c} 1 \\ 0 \\ \vdots \\ 0 \end{array}\right],~~~ \evec_2 = \left[\begin{array}{c} 0 \\ 1 \\ \vdots \\ 0 \end{array}\right],~~~ \ldots,~~~ \evec_n = \left[\begin{array}{c} 0 \\ 0 \\ \vdots \\ 1 \end{array}\right] \end{equation*}
so that
\begin{equation*} T(\evec_j) = \left[\begin{array}{rrrr} \vvec_1 \amp \vvec_2 \amp \ldots \amp \vvec_n \end{array}\right] ~\evec_j = \vvec_j\text{.} \end{equation*}
This means that the \(j^{th}\) column of \(A\) is found by evaluating \(T(\evec_j)\text{.}\) We record this fact in the following proposition.
