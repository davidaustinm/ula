1. We have the augmented matrix
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 2 \amp 4 \amp 0 \\ -2 \amp 1 \amp -3 \amp 0 \\ 3 \amp 1 \amp 7 \amp 0 \\ \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 2 \amp 0 \\ 0 \amp 1 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \\ \end{array}\right] \end{equation*}
    so that \(\xvec = x_3\threevec{-2}{-1}{1}\text{.}\) The solution space is a line that passes through the origin.
  2. Now we have
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 2 \amp 4 \amp -3 \\ -2 \amp 1 \amp -3 \amp -4 \\ 3 \amp 1 \amp 7 \amp 1 \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 2 \amp 1 \\ 0 \amp 1 \amp 1 \amp -2 \\ 0 \amp 0 \amp 0 \amp 0 \end{array}\right] \end{equation*}
    so that \(\xvec=\threevec{1}{-2}{0}+x_3\threevec{-2}{-1}{1}\text{.}\) This is a line through \(\threevec{1}{-2}{0}\) parallel to \(\threevec{-2}{-1}{1}\text{.}\) Notice that this line is parallel to the line in part a.
  3. Notice that \(A(\xvec_h + \xvec_p) = A\xvec_h + A\xvec_p = \zerovec + \bvec = \bvec\text{.}\) This shows that if we add a vector from the solution space to \(A\xvec=\zerovec\) to a vector from the solution space to \(A\xvec=\bvec\text{,}\) we obtain a vector in the solution space to \(A\xvec=\bvec\text{.}\)
