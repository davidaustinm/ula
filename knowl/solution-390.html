Solution 7.2.4.9.1.

  1. \begin{align*} f_x \amp {}={} y - \frac2{x^2} = 0\\ f_y \amp {}={} x - \frac2{y^2} = 0 \end{align*}
    Also, \(f_{xx} = \frac4{x^3}\text{,}\) \(f_{xy} = 1\text{,}\) and \(f_{yy} = \frac4{y^3}\) so that the Hessian matrix is \(\begin{bmatrix} 1/2 \amp 1 \\ 1 \amp 1/2 \end{bmatrix}\text{.}\) The eigenvalues are \(-1/2\) and \(3/2\) so this matrix is indefinite and the critical point is neither a local maximum nor minimum.
  2. \begin{align*} f_x \amp {}={} 4x^3 - 4y = 0\\ f_y \amp {}={} 4y^3 - 4x = 0 \end{align*}
    so \(x=y^3\) and \(y=x^3\text{.}\) Therefore, \(x=x^9\) so \(x=y=0\text{,}\) \(1\text{,}\) or \(-1\text{.}\)
    Also, \(f_{xx} = 12{x^2}\text{,}\) \(f_{xy} = -4\text{,}\) and \(f_{yy} = 12{y^2}\) so that the Hessian matrix is \(\begin{bmatrix} 12x^2 \amp -4 \\ -4 \amp 12y^2 \end{bmatrix}\text{.}\)
    At \((0,0)\text{,}\) the Hessian is indefinite so this critical point is neither a local maximum nor minimum. At both \((1,1)\) and \((-1,-1)\text{,}\) it is positive definite so the critical points are local minima.
