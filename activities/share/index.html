Activities for teaching linear algebra

David Austin (austind at gvsu dot edu)

I taught a one-semester introduction to linear algebra in Fall 2018 using the text Understanding Linear Algebra. In addition to the preview activities in the text, our class meetings (three 50-minute meetings per week) were largely driven by the set of activities on this page.

Anyone is welcome to use these for educational purposes. All the source (latex and image) files are included in the .zip file below along with individual links to each activity in .pdf format.

There are undoubtedly some typos and (hopefully only small) errors so buyer beware. Feel free to send questions, comments, and/or complaints to me.
